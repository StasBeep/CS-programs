# CS-programs
Программы на C#

# Папки

`Matrix` - Назовем допустимым преобразованием матрицы перестановку двух строк и
двух столбцов. Дана действительная квадратная матрица порядка n. С помощью
допустимых преобразований добиться, чтобы один из элементов матрицы, обладающий наибольшим по модулю значением, располагался в левом верхнем
углу матрицы.
